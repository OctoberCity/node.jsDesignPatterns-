### 复用和分解 （重点）
```
通道1===                                  ====》通道a  
        ====》                        ====
              复用 ==》共享通道==》 分解
        ====》                        ====
通道2 ===                                 ====》通道b 
```
将多个流合并在一起，成为一个流，以便使用其传输数据的模式称为复用（multiplexing）,其由装置mux完成，而将共享通道中的数据重新构建原始流的操作叫分解,其由装置demux完成。
